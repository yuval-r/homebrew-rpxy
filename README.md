# homebrew-rpxy

Homebrew tap for [rpxy](https://rpxy.io) - a simple and ultrafast HTTP reverse proxy written in Rust.

## Installation

This formula installs pre-built macOS binaries, so no build dependencies are required.

## Installation
```bash
brew tap yuval-r/rpxy
brew install rpxy
```

## Usage
```bash
rpxy --help
```

## Updating
```bash
brew upgrade rpxy
```
